Introduction¶

Quick description: Project's objective is to perform sentiment analysis on news stories about specified stocks, utilizing Finhub to source the news and ChatGPT to perform the analysis. The number of stocks and news analyzed through ChatGPT is kept deliberately small to avoid costs as this is for demonstration purposes. Ideally the entire news set returned by Finhub should be used, however such an amount would be markedly more expensive to process through ChatGPT.¶

Step 1.¶

Import all libraries¶

Step 2.¶

Fetch financial news, using Finhub for the following stocks: Apple, Exxon Mobil, Google, Nvidia, Rockwell Automation, Facebook, Chevron and Target¶

Step 3.¶

Send the news stories to ChatGPT for analysis, which will return a sentiment analysis between -1 and 1, a confidence index and the reasoning behind its sentiment score.¶

Step 4.¶

Calculate the Sentiment and Confidence average per stock as well as using ChatGPT to generate a summary of all reasonings per stock to produce a final summary for the score given.¶

Step 3 - Performing Sentiment Analysis with Chat GPT 4.0¶

In [18]:
for index, row in GPT_news.iterrows():
    # Extract the headline
    headline = row['headline']
    print("Processing headline: "+headline)
    # Send the headline to the API
    #AAPL , XOM , GOOG , NVDA , ROK , META , CVX , TGT
    response = client.chat.completions.create(
        model="gpt-4",
        messages=[
            {"role": "system", "content": f"You are a stock sentiment analysis assistant. You will analyze sentiment in headlines for the following stocks: {stock_list} "},
            {"role": "system", "content": "Provide answers in JSON format with no line space included. THe JSON headers will be sentiment, confidence and reasoning. You will provide exactly 1 JSON row in response to each question and will adhere to the mentioned heaaders"},
            {"role": "user", "content": "The following headline is an example of a positive story for Walmart and a negative headline for Target, learn from it: Target, Kohl’s and Best Buy each reported disappointing results in their most recent quarters, but Walmart, Dick’s Sporting Goods and Abercrombie & Fitch posted strong sales."},
            {"role": "user", "content": f" 1. What is the sentiment of this headline in regards to the stocks in it, use only numbers the range of -1 to 1 to describe sentiment, -1 for negative, 0 for neutral, 1 for postive; dont use letters: '{headline}'? 2.Provide the confidence level for that sentiment rating with values between 0 and 1 3.Provide the reasons as to why you chose that sentiment rating for that stock"}
        ]
    )
    js = json.loads(response.choices[0].message.content)

    df_js=json_normalize(js)

    GPT_news.at[index, 'sentiment'] = df_js['sentiment'].values[0]
    GPT_news.at[index, 'confidence'] = df_js['confidence'].values[0]
    GPT_news.at[index, 'reasoning'] = df_js['reasoning'].values[0]
